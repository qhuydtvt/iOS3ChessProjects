
How to build the .so libraries in this 'jni' folder:
=====================================================

$ ndk-build

The above command will create the necessary .so files:
../libs/armeabi/libAI_MaxQi.so
../libs/armeabi/libReferee.so
../libs/armeabi-v7a/libAI_MaxQi.so
../libs/armeabi-v7a/libReferee.so
../libs/mips/libAI_MaxQi.so
../libs/mips/libReferee.so
../libs/x86/libAI_MaxQi.so
../libs/x86/libReferee.so
